## git-bug show

Display the details of a bug.

### Synopsis

Display the details of a bug.

```
git-bug show [<id>] [flags]
```

### Options

```
  -f, --field string   Select field to display. Valid values are [author,authorEmail,createTime,id,labels,shortId,status,title]
  -h, --help           help for show
```
