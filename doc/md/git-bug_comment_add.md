## git-bug comment add

Add a new comment.

### Synopsis

Add a new comment.

```
git-bug comment add [<id>] [flags]
```

### Options

```
  -F, --file string      Take the message from the given file. Use - to read the message from the standard input
  -m, --message string   Provide the new message from the command line
  -h, --help             help for add
```
