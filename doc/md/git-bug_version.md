## git-bug version

Show git-bug version information.

### Synopsis

Show git-bug version information.

```
git-bug version [flags]
```

### Options

```
  -n, --number   Only show the version number
  -c, --commit   Only show the commit hash
  -a, --all      Show all version informations
  -h, --help     help for version
```
